# Rock Paper Pro

A polished Rock-Paper-Scissors game built with Next.js, React, and TypeScript.

## Features

- Keyboard-friendly gameplay with quick choice hotkeys.
- Animated UI with sound effects.
- Adaptive AI that learns from recent player history.
- Built-in score tracking and smooth round transitions.

## Tech Stack

- Next.js
- React
- TypeScript
- Tailwind CSS

## Getting Started

### Prerequisites

- Node.js 18+ (recommended: latest LTS)
- npm

### Install

```bash
npm install
```

### Run in Development

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

### Production Build

```bash
npm run build
npm run start
```

## Available Scripts

- `npm run dev` - start local dev server.
- `npm run build` - create production build.
- `npm run start` - run production server.
- `npm run lint` - run ESLint.
- `npm run format` - run Prettier.
